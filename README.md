# Webapp Skills Plugin for GitHub Copilot CLI

A curated collection of 9 webapp development skills for GitHub Copilot CLI, powered by the [claude-skills](https://github.com/alirezarezvani/claude-skills) repository.

## 📦 Included Skills

1. **Frontend** - Senior Frontend Engineer (React, Vue, Angular, TypeScript, Tailwind)
2. **Backend** - Senior Backend Engineer (Node.js, Express, Python, APIs, Database)
3. **Fullstack** - Senior Fullstack Engineer (Full web application development)
4. **DevOps** - Senior DevOps Engineer (CI/CD, deployment, infrastructure)
5. **API Design** - API Design Reviewer (REST, GraphQL, best practices)
6. **Database** - Database Designer (Schema, migrations, optimization)
7. **Testing** - Testing & QA Engineer (Unit, integration, e2e)
8. **Docker** - Docker Development (Containerization, Docker Compose)
9. **CI/CD** - CI/CD Pipeline Builder (GitHub Actions, automation)

## 🚀 Installation

```bash
copilot plugin install ~/copilot-webapp-skills-plugin
```

## 💡 Usage

After installation, skills will appear in the interactive Copilot CLI skills menu.

1. Start Copilot: `gh copilot` or `copilot`
2. Press `/` to open menu
3. Select "Skills" to see available skills
4. Select a skill to activate its context
